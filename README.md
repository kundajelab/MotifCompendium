# MotifCompendium
The MotifCompendium package is framework for clustering, annotating, and analyzing motifs efficiently at large scales.

A wiki about key concepts is currently being developed.

For starter code, please go through the tutorials. **It is highly recommended to at least go through tutorials 1 and 2**.

## Installation
1. Create a conda environment by either running `conda env create -f environment.yml` or `conda env create -f environment_gpu.yml`.
2. Activate the newly create conda environment.
3. In this directory, run `pip install -e .`.
**This package is being actively developed, so please git pull the package before each time you use it.**
